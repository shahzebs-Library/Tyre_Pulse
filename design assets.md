Yes. This is one area where I would **not** rely on AI-generated SVGs alone.

For a commercial SaaS like Tyre Pulse, the illustrations should look premium, consistent, and branded.

### My recommendation

Don't use random SVGs from different sources.

Build an **Illustration System**.

Use:

* **Figma** as the master design source.
* **SVG** for scalable illustrations.
* **Lottie** (`.json`) for animated illustrations.
* **Rive** for interactive animations (premium feel).
* **SVGO** to optimize SVG size.
* **Iconify** + **Lucide** or **Phosphor Icons** for all icons.
* **Framer Motion** for subtle page animations.
* **React SVG** components so colors automatically adapt to Light/Dark themes.

### Where SVGs should be used

Instead of only empty-state graphics, create branded illustrations for:

* Dashboard
* Fleet
* Tyres
* Inventory
* Accident Management
* Workshop
* Purchase Orders
* AI Assistant
* Reports
* Notifications
* No Data
* Loading
* Error
* Offline Mode
* Maintenance
* Approval Workflow
* Search Empty State
* 404
* 500 Error
* Success Pages

All should follow the same illustration style.

### Add depth

Your SVGs shouldn't be flat clipart.

They should include:

* Soft gradients
* Layered shadows
* Depth
* Professional color palette
* Enterprise industrial style
* Fleet, trucks, tyres, mechanics, workshops, inspections
* Consistent stroke width
* Consistent corner radius
* Brand colors
