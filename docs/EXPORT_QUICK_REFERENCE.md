# TyrePulse Export Quick Reference

**One-page cheat sheet for developers** | v2.0.0

---

## 🚀 Quick Start

### Import
```javascript
import { exportToExcel, exportToPdf, exportToPptx } from '../lib/exportUtils'
```

### Basic Excel Export
```javascript
exportToExcel(
  rows,                              // Array of objects
  ['asset_no', 'brand', 'cost'],    // Column keys
  ['Asset No', 'Brand', 'Cost'],    // Column headers
  'Fleet_Report_2026-07-02'         // Filename (no extension)
)
```

### Basic PDF Export
```javascript
exportToPdf(
  rows,
  [
    { key: 'asset_no', header: 'Asset No' },
    { key: 'cost', header: 'Total Cost' },
    { key: 'risk_level', header: 'Risk' }
  ],
  'Fleet Report',                    // Title on page
  'Fleet_Report_2026-07-02',         // Filename
  'landscape',                       // 'portrait' or 'landscape'
  'Your Company',                    // Company branding
  { currency: 'SAR' }                // Options
)
```

### Basic PowerPoint Export
```javascript
exportToPptx({
  company: 'Your Company',
  period: 'July 2026',
  currency: 'SAR',
  totalVehicles: 50,
  totalTyres: 300,
  criticalTyres: 12,
  openActions: 8,
  // ... 50+ data fields (see EXPORT_GUIDE.md)
}, 'Fleet_Report_2026-07-02')
```

## 🔧 Common Patterns

### Filter Data Before Export
```javascript
// Limit to first 5,000 records (prevent OOM)
const exportData = allRows.slice(0, 5000)

// Warn user if truncated
if (allRows.length > 5000) {
  console.warn(`Export limited to 5,000 of ${allRows.length} records`)
  toast.info(`Exported first 5,000 records. Total: ${allRows.length}`)
}
```

### Check for Valid Data
```javascript
// Validate before export
if (!rows || rows.length === 0) {
  toast.error('No data to export')
  return
}

if (!columns || columns.length === 0) {
  toast.error('No columns selected')
  return
}
```

### Async Export with Loading State
```javascript
const [exporting, setExporting] = useState(false)

const handleExport = async (format) => {
  setExporting(true)
  try {
    if (format === 'xlsx') {
      exportToExcel(rows, cols, headers, filename)
    } else if (format === 'pdf') {
      exportToPdf(rows, columns, title, filename, 'landscape', company)
    } else if (format === 'pptx') {
      await exportToPptx(data, filename)
    }
    toast.success(`${format.toUpperCase()} downloaded`)
  } catch (err) {
    toast.error(`Export failed: ${err.message}`)
  } finally {
    setExporting(false)
  }
}
```

### Auto-Detect Numeric Columns
```javascript
// Automatically summarize cost columns
const numCols = columns.filter(k => {
  const sample = rows.slice(0, 60).map(r => r[k])
  const nums = sample.filter(v => !isNaN(parseFloat(v)))
  return nums.length / sample.length > 0.75
})

// Include totals + averages in summary
numCols.forEach(k => {
  const total = rows.reduce((sum, r) => sum + parseFloat(r[k] || 0), 0)
  const avg = total / rows.length
  console.log(`${k}: Total ${total}, Avg ${avg}`)
})
```

### Handle Currency Formatting
```javascript
// Format numbers as currency
const formatCurrency = (value, currency = 'SAR') => {
  const num = parseFloat(value) || 0
  return new Intl.NumberFormat('en-US', {
    style: 'currency',
    currency,
    minimumFractionDigits: 0,
  }).format(num)
}

// Use in export
exportToExcel(
  rows.map(r => ({ ...r, cost: formatCurrency(r.cost, 'SAR') })),
  columns,
  headers,
  filename,
  'Data',
  { currency: 'SAR' }
)
```

## 🐛 Quick Troubleshooting

| Problem | Cause | Fix |
|---------|-------|-----|
| **Blank PDF** | No rows | Check `rows.length > 0` before export |
| **Excel won't open** | NaN/Infinity values | Use `Math.isFinite()` check; filter bad data |
| **PPTX corrupted** | Empty chart labels | Use `cleanSeries()` wrapper on all charts |
| **Slow export** | 100k rows | Limit to `rows.slice(0, 10000)` |
| **Missing company name** | Not passed to function | Add `company` param to export function |
| **Risk colors missing** | No `risk_level` column | Column key must exactly match column in data |
| **Page breaks ugly** | autoTable bug | Manually control `startY`; test page breaks |

## ⚡ Performance Tips

| Action | Impact | Best Practice |
|--------|--------|---|
| **Paginate large exports** | -70% memory | Limit to 5k–10k records |
| **Use specific columns** | -40% query time | Select only needed fields from DB |
| **Cache SVG diagrams** | -60% PDF gen time | Store converted PNGs in memory |
| **Async generate** | 0ms UI freeze | Use `await exportToPptx()` |
| **Server-side filtering** | -85% network | Apply filters in Supabase query |
