# Screenshots

Put your screenshot images in this folder using the exact file names below.
Once a file exists, it shows up automatically in FEATURES.md (Screenshots section).

Take these on a logged in device for the best result.

Suggested shots and file names:

| File name | What to capture |
|-----------|-----------------|
| dashboard.png | Web dashboard with KPIs |
| tyre-records.png | Tyre Records list with filters |
| accident-claims.png | An accident detail with Claim and Recovery tab |
| accident-activity.png | The Activity tab showing who changed what |
| upload-preview.png | Upload step with File Preview and header row picker |
| upload-quality.png | The Data Quality report before approve |
| reports.png | A finished report or Executive Report |
| analytics.png | Any analytics page with charts |
| mobile-inspection.png | Mobile tyre diagram with the tap popup |
| mobile-scan.png | Mobile scanner screen |
| mobile-accident.png | Mobile accident report or claims panel |

Tips:
- Use PNG. Keep width around 1400px for web shots and the real phone size for mobile shots.
- Avoid showing real personal data if you plan to share the file publicly.
